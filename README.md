# Predicting Sea Level Changes on Long Island using Machine Learning Algorithm

## Repository Structure

```bash
├── src/                         # Scripts for evaluating models
   ├── gauss.ipynb               # Training, testing and evaaluating the Gaussian Process Model
   ├── linear_regression.ipynb   # Training, testing and evaaluating the Linear Regression Model
   ├── lstm.ipynb                # Training, testing and evaaluating the lstm model
   ├── random_forest.ipynb       # Training, testing and evaaluating the Random Forest
   ├── replot.ipynb              # Plotting the Random Forest Gini Impurity Bar Chart
   ├── replot2.ipynb             # Plotting the XGBoost SHAP value Bar Chart
   ├── rf_hyperparameter.ipynb   # Calcuating Random Forest Gini Impurity value
   ├── rnn.ipynb                 # Training, testing and evaaluating the RNN Model
   ├── rnn_hyperparameter.ipynb  # Calculating the hyperparameters for the RNN model
   ├── xg_hyperparamter.ipynb    # Calculating XGBoost SHAP values
   └── xgboost.ipynb              # Training, testing and evaaluating the XGBoost Model
```
